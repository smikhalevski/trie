@smikhalevski/trie

trie

The compact trie data structure.

npm install --save-prod @smikhalevski/trie

Usage

⚠️ API documentation is available here.

import {TrieMap} from '@smikhalevski/trie';

const trieMap = new TrieMap();

trieMap.set('foo', 123);

trieMap.get('foo'); // → 123

trieMap.search('aaafoobbb', 3);
// → {key: 'foo', value: 123, …}
